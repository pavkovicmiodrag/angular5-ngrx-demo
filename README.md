# Dashboard

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 1.7.2
It is small demo project to demonstrate angular 5 application with usage of ngrx library (store, effects, reducer, actions and entity).
Application  list diagnoses (name and description) and enable user to do CRUD operations on them. Angular material 5 is also used with theming to support styling.
## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
